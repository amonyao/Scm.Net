【项目说明】
此项目主要用于表达〖数据存储对象〗（Data Access Object）映射类，不参与具体的业务逻辑。
当有多个项目需要使用相同的DAO时，可以引用此项目。

【注意事项】
1、此模块可以根据需要单独存在，或者与Server项目整合；
2、同一个类唯一映射到一个数据库表格，用于对外呈现表格信息；
3、多个类可以映射到同个数据表格，但需要注意数据库字段的默认取值及为空问题，不建议使用此方法。
4、每个Dao类均可重写PrepareCreate、PrepareUpdate、PrepareDelete方法，用于CUD操作的事前处理，如生成ID、时间处理等。【项目说明】

【项目依赖】
1、SqlSugarCore：数据ORM工具；
2、Scm.Common.Uid：ID生成器，包含雪花ID、序列ID、格式ID等多种形式。
